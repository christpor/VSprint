# 🚀 VSprint

> 🤖 **My First AI Learning Project** — built entirely with the help of AI. This is where the journey started.

---

## 💡 What is VSprint?

VSprint is a web app for developers that uses AI to help write code. Talk to an AI, get code back, and see it run in a live preview — instantly.

---

## ✨ Features

- 🔐 **Easy Login** — Sign in with your email or Google account.
- 🤖 **AI Coding** — Powered by Gemini to help you write functions and components.
- ⚡ **Live Preview** — See your code run instantly on screen.
- 🕓 **History** — The app remembers your past chats so you can revisit them.
- 🎮 **Demo Mode** — Try the app without creating an account first.
- 🌙 **Dark Mode** — A dark theme that is easy on your eyes.
- 🎬 **Smooth Motion** — Cinematic animations that make the app feel premium.

---

## 🛠️ Tech Stack

| Category | Technology |
|---|---|
| Frontend | React 19, TypeScript, Vite, Tailwind CSS |
| Backend | Node.js, Express |
| Database & Auth | Supabase |
| AI Engine | Google Gemini API |
| Animations | Framer Motion |
