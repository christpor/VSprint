# 🎯 Project Overview: VSprint

## Core Mission
VSprint is an elite, high-density coding coach designed for rapid skill acquisition. It prioritizes "The Why" behind "The How" and uses modern AI models to deliver instant, visually stunning code solutions.

## Vision
To build a world where anyone can master complex technical concepts through cinematic, interactive learning experiences. VSprint doesn't just give answers; it builds developers through senior-level insights and active mastery drills.

## Core User Flows
1. **The Ask:** User inputs a coding problem or request.
2. **The Analysis:** VSprint decomposes the problem into architectural logic and senior insights.
3. **The Solution:** Instant generation of production-ready HTML/CSS/JS with a Live Preview.
4. **The Mastery:** Targeted drills to solidify understanding.

## Target Audience
- Beginners looking for deep understanding rather than copy-pasting.
- Developers looking for rapid prototyping with high-fidelity UI.
- Learners who thrive on visual, cinematic feedback.

## Out-of-Scope Boundaries
- VSprint is NOT a full IDE; it is a coding coach and component generator.
- Complex multi-file project management is currently handled by the user/CLI; the web UI focuses on atomic mastery.
